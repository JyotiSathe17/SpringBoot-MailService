# SpringBoot-MailService

## Description
A **Spring Boot application** that sends emails using **Java Mail API**.  
Supports sending emails with **CC, BCC, subject, body, and attachments**.

---

## Features
- Send email using SMTP
- Support for:
  - From
  - To
  - CC (Carbon Copy)
  - Subject
  - Email Body (Text/HTML)
  - Attachments
- REST API for sending emails
- Exception handling for email failures

---

## Technologies Used
- Java
- Spring Boot
- Java Mail API
- Maven
- Postman (for API testing)

---

## API Endpoint
- **POST** `/sendMail`
  - Sends an email with given details
